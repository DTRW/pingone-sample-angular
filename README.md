# PingOne for Customers Angular Samples
This repository contains code samples that demonstrate how to use PingOne for Customer SignIn\SignOut, Registration and Password Recovery in Angular projects

## Samples:
- [**OpenID Connect sample**](pingone-angular-openidconnect-sample) - User SignIn and SignOut using OIDC
- [**User Registration sample** ](pingone-angular-registration) - User Registration and Password Recovery using PingOne Management API
